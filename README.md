# Debian Server Scripts & Config Collection

Det här repot innehåller en samling av scripts, konfigurationsfiler och små verktyg som jag använder i samband med drift, installation och underhåll av Debian‑servrar. Syftet är att ha ett centraliserat och versionshanterat arkiv över sådant jag ofta behöver i mina miljöer.

## Innehåll

I repot finns exempelvis:

- **Shell-skript** för administration, automationsjobb och felsökning  
- **Konfigurationsfiler** (t.ex. `.nanorc`, `.vimrc`, systemd‑units)  
- **Exempelfiler** för nätverk, brandvägg, tjänsteinstallationer  
- **Verktyg/alias** jag använder för att underlätta vardagligt terminalarbete  
- **Notes/snippets** jag vill ha nära till hands när jag sätter upp nya maskiner

Strukturen är tänkt att vara enkel och modulär, så man snabbt hittar det man söker.

## Målgrupp

Det här repot är i första hand för **mig själv**, men kan vara användbart för:

- Systemadministratörer som jobbar med Debian‑servrar  
- DevOps-ingenjörer eller studenter inom Linux/IT‑drift  
- Personer som vill se hur någon annan strukturerar sin servermiljö  
- Alla som vill ha exempel på praktiska konfigurationsfiler eller scripts
